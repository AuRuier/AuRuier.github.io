hiberste 方言问题

在 application.yml 中的 hibernate 原设置为database-platform: org.hibernate.dialect.MySQL8InnoDBDialect

然而根据错误提示信息来看，Hibernate 无法解析这个方言。

故将方言修改为：database-platform: org.hibernate.dialect.MySQL8Dialect

mysql 连接问题

springboot，mysql，hiberate 依赖驱动等等版本需要兼容

要求：

  • 存在数据库advice_collection_db2且不存在表 @Table(name = “ADVICE“)；
  • 输入数据类型与设置实体一致

test：

INSERT INTO advice (name, title, email, detail, date_created) VALUES (‘John’, ‘Sample Advice’, ‘john@example.com‘, ‘This is a sample advice.’, NOW());
